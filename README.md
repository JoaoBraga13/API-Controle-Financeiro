======== OBJETIVO ========

O principal objetivo desse projeto é colocar em prática todos estudos aprendidos sobre Typescript, ferramentas de API e Bancos de dados.
Este projeto é focado no Backend, visando abordar conceitos como criação de CRUD, arquitetura de projeto MVC, testes e mais.


======== FERRAMENTAS E TECNOLOGIAS ========

- Typescript
- NodeJS
- ExpressJS
- MySQL
- TypeORM
- ESlint
- Prettier
- Bcrypt
- Insomnia
